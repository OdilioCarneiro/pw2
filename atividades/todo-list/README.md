Crie um componente React que permita ao usuário:
Adicionar novas tarefas via um formulário com um campo de input e botão
de submit

Marcar tarefas como concluídas ou não concluídas com um evento de clique
(ex.: checkbox ou botão)

todas, com base em um filtro (ex.: botões "Todas", "Pendentes",
"Concluídas").
